About Me

I'm Edrean Supremo, a Computer Engineering student passionate about building systems that live at the intersection of hardware and software. Whether I'm designing digital circuits with Verilog, programming microcontrollers for IoT solutions, or architecting full-stack web applications, I approach every challenge with curiosity and precision.

My journey spans from low-level hardware design to high-level software architecture. I've built autonomous robots, deployed smart classroom systems, and created web platforms that serve real users. I believe great engineering is about understanding the full stack — from transistors to user interfaces.

Beyond the technical work, I'm deeply committed to sharing knowledge with others. I've spoken at workshops on electronics and portfolio building, and I thrive in competitive environments like hackathons and programming contests. Every project is a chance to push boundaries and learn something new.
